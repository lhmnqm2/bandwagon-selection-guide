## 搬瓦工服务器选哪个好：先搞清楚三条线路的区别

在看套餐之前，先把线路搞懂，不然选错了再迁移是一件头疼的事。

**CN2 GT（已停售，不再推荐）**
电信骨干网的中端版本，部分路段走 CN2，部分走普通直连。搬瓦工官方已停止销售这个线路的套餐，现有用户续费尚可，新人直接跳过。

**CN2 GIA**
全程走中国电信的企业级优化线路，去程和回程都不绕路。晚高峰丢包率通常很低，延迟也比普通线路稳。这是搬瓦工目前主推的核心线路，从美国洛杉矶到国内，电信走得最顺。

**CN2 GIA-E（ECOMMERCE 版）**
GIA 的进阶版。三网直连——电信 CN2 GIA，联通 AS9929 或直连，移动 CMIN2 高端线路。三个运营商的用户晚高峰体验都相对均衡，是目前搬瓦工套餐里性价比最高的系列。

顺便说一句，CMIN2 是移动自己的骨干优化线路，比普通 CMI 更稳，这两年才逐步普及。

## 搬瓦工服务器选哪个好：按使用场景推荐

这是核心。别看配置选套餐，要看用途。

### 场景一：学习 Linux / 跑小项目 / 预算有限

**选 KVM 套餐，$49.99/年**

这档套餐的线路是普通 CN2，不是 GIA，速度和稳定性不如 GIA-E，但价格在搬瓦工里算最低。如果你的需求是有台海外服务器折腾折腾，对连国内速度没极致要求，这个足够。

或者等限量版补货。BiggerBox Pro / MegaBox Pro 补到的话，同样的价格能拿到 CN2 GIA 线路，性价比完全不是一个量级。

### 场景二：建站，受众在国内

**首选 CN2 GIA-E 套餐，$169.99/年（或季付 $49.99 先试试）**

支持 12 个以上机房自由迁移。买进去后，可以在 DC6、DC9、日本软银、荷兰 AS9929 等机房之间切换，找到最适合你目标用户的节点。对于需要国内用户访问流畅的网站，这条线路是目前搬瓦工里经过验证最稳的选择。

三网表现均衡，电信、联通、移动晚高峰基本不会出现大规模丢包。

### 场景三：日常远程办公 / SSH 开发 / 对延迟敏感

**CN2 GIA-E 套餐（同上）或考虑日本大阪/东京节点**

CN2 GIA-E 买了之后可以迁到日本软银机房，延迟会比美国洛杉矶低一截。如果你主要在广东或华南地区，香港 CN2 GIA 才是真正的"体感差距明显"，但价格门槛高多了。

### 场景四：追求极致低延迟，预算充足

**香港 CN2 GIA，$89.99/月起**

物理距离近，三网全程 CN2 GIA。广东方向延迟可以低到个位数毫秒。确实是体验上的天花板。

价格也是天花板。一个月 $89.99，一年将近 $900，不是每个人都用得起。

说实话，对大多数人来说 CN2 GIA-E 已经够了，香港套餐是那种"你知道有更好的，但不一定需要它"的东西。

## 搬瓦工机房怎么选

买了 CN2 GIA-E 套餐之后，下一步就是选机房。常见问题是：买了套餐默认给的机房不一定最适合你，后台可以免费迁移。

**推荐大多数国内用户的起点：DC6 或 DC9**

两者都在洛杉矶，线路都是三网 CN2 GIA，日常表现相近。DC6 的带宽冗余通常更大一些；DC9 所有出站流量会先经过中国电信网络再转给其他运营商，两者各有场景。

**想试试亚太线路：日本软银 JPOS_1**

白天表现很不错，延迟低于美西机房。晚高峰会有一定波动，取决于地区和运营商。如果你在华南方向，可以先测一下日本软银的测试 IP 再决定是否迁移。

**联通用户的额外选择：荷兰 AS9929 或纽约 USNY_8**

AS9929 是联通的高端直连线路，在荷兰机房和部分美国机房上有，联通用户在这条线路上体感会比 CN2 GIA 好一些。纽约机房有 CN2 GIA + CMIN2 + CUP 三网优化，适合业务偏向北美的用户。

## 退款和 IP 被封这两件事

**退款**：新用户购买后 30 天内支持无理由全额退款，后台直接申请，自动处理，条件是需满足一定使用限制（主要是流量用量不能太多）。这个算是比较实在的保障——如果买了发现不合适，退得回来。

**IP 被封**：这是搬瓦工用户被提得最多的一个问题。如果 IP 地址被 GFW 封锁，更换 IP 收费 $8，不便宜。流量用量少于 10% 的话可以申请退款（换句话说，买了没用多少就遇到封 IP 的，不用慌，可以选择退）。

实际用下来，CN2 GIA-E 的 IP 被封频率相对 KVM 套餐要低一些，但不是不会发生。如果你所在地区封 IP 比较频繁，选套餐的时候要把这个成本考虑进去。

## FAQ

**Q：搬瓦工 KVM 套餐和 CN2 套餐有什么区别？**

KVM 是架构类型，这里说的"KVM 套餐"特指搬瓦工最低价那一档，走的是普通线路（非 CN2 优化）。CN2 套餐也是 KVM 架构，但走的是中国电信的 CN2 骨干网，连国内速度更快更稳，价格一样都是 $49.99/年。同等预算下，CN2 套餐性价比比纯 KVM 套餐好。

**Q：CN2 GIA 和 CN2 GIA-E 的区别是什么？**

CN2 GIA 通常指只对电信做了优化的线路；CN2 GIA-E（ECOMMERCE 版）则是三网优化，电信走 CN2 GIA，联通走 AS9929 或 CUP，移动走 CMIN2。对于电信以外的用户来说，GIA-E 的体验会明显好于普通 GIA。

**Q：搬瓦工支持支付宝付款吗？**

支持，结账时选择支付宝即可，这对国内用户来说算是比较方便的。

**Q：限量版套餐缺货了怎么办？**

**Q：买了套餐之后想换机房怎么操作？**

登录 KiwiVM 后台，找到 Migrate 选项，选择目标机房，确认后自动迁移，整个过程通常几分钟内完成。
